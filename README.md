# Mahalle Bazlı Duyuru Uygulaması

Bu proje Flutter ve MongoDB kullanılarak geliştirilmiş konum tabanlı mahalle duyuru uygulamasıdır.

## Özellikler
- Kullanıcı konumundan mahalle tespiti (Geolocator + Geocoding)
- Mahalleye göre duyuru listeleme
- Tarih/saat sıralaması
- Duyuru ekleme ve filtreleme
- Kullanıcı kayıt ve giriş sistemi

## Kullanılan Teknolojiler
- Flutter
- MongoDB
- Node.js API
- Geolocator
